# RAG Demos - 检索增强生成示例

## 简介

本模块包含RAG（Retrieval-Augmented Generation）的实现示例，学习如何结合检索和生成技术。

## 包含的示例

### 1. Basic RAG
- 文档加载和切分
- 向量化和存储
- 检索和生成流程

### 2. Advanced RAG
- 混合检索策略
- 重排序优化
- 上下文压缩

## 使用方式

```bash
cd rag-demos
mvn spring-boot:run
```

## 学习要点

- 理解RAG的工作原理
- 掌握文档处理和切分技巧
- 学习检索策略的优化方法
- 了解如何提升生成质量
