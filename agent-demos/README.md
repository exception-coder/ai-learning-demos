# Agent Demos - AI Agent示例

## 简介

本模块包含AI Agent的设计和实现示例，学习如何构建智能代理系统。

## 包含的示例

### 1. Simple Agent
- 基础Agent架构
- 工具调用机制
- 任务规划和执行

### 2. Multi-Step Agent
- 多步推理
- 状态管理
- 错误处理和重试

## 使用方式

```bash
cd agent-demos
mvn spring-boot:run
```

## 学习要点

- 理解Agent的核心概念
- 掌握工具调用的实现方式
- 学习任务规划和分解策略
- 了解Agent的状态管理机制
