# LLM Demos - 大语言模型示例

## 简介

本模块包含各类大语言模型API的使用示例，帮助学习和理解如何调用和使用LLM。

## 包含的示例

### 1. OpenAI Demo
- GPT-3.5/GPT-4 API调用
- 流式响应处理
- 对话上下文管理

### 2. DeepSeek Demo
- DeepSeek API使用
- 中文对话优化
- 成本控制策略

### 3. Ollama Demo
- 本地模型部署
- 离线使用场景
- 模型切换和管理

## 使用方式

```bash
cd llm-demos
mvn spring-boot:run
```

## 配置说明

在 `application.yml` 中配置相应的API Key和模型参数。

## 学习要点

- 理解LLM的调用流程
- 掌握流式响应的处理方式
- 学习对话上下文的管理策略
- 了解不同模型的特点和适用场景
