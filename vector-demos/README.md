# Vector Demos - 向量数据库示例

## 简介

本模块包含向量数据库的使用示例，学习如何进行向量存储和语义搜索。

## 包含的示例

### 1. Qdrant Demo
- 向量存储和检索
- 相似度搜索
- 过滤条件使用

### 2. Milvus Demo
- 大规模向量管理
- 索引优化
- 性能调优

## 使用方式

```bash
cd vector-demos
mvn spring-boot:run
```

## 前置条件

需要先启动相应的向量数据库服务（如Qdrant、Milvus）。

## 学习要点

- 理解向量化的概念
- 掌握向量存储和检索的方法
- 学习相似度计算原理
- 了解向量索引的优化策略
